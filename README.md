# Musica Viva Tours — Geography of Sound

The official one-page site for **Musica Viva Tours** (musicavivatours.com): private concerts by
cellist Max Beitan, staged within heritage palaces and noble estates, curated by the Beitan Music
Foundation.

## What's here

```
index.html      The complete site (single page, no build step)
support.js       Runtime that renders the page
assets/          Photography + hero video (only the files the page uses)
```

Open `index.html` in any modern browser, or serve the folder statically.

## Deploy to GitHub Pages

1. Push this folder to a repository.
2. In **Settings → Pages**, set the source to the `main` branch, root (`/`).
3. The site publishes at `https://<user>.github.io/<repo>/`.

No bundler, package manager, or server runtime is required — it is a static site.
